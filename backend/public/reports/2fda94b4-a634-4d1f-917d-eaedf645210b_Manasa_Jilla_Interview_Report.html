Rapid Fire — DSA Theory
Check if a number is a power of 2 — most efficient approach
Proposed: square root approach — compute square root, check if the result is an integer — O(1) but not the optimal bit manipulation solution When prompted for bit manipulation: identified that a power of 2 in binary has exactly one set bit with all remaining bits as zeros (e.g. 16 = 10000) Could not cleanly articulate the single-line bit trick: n > 0 && (n & (n-1)) == 0 Direction was correct; precise formulation was incomplete
~ Partial — Bit manipulation direction correct; optimal one-liner not articulated (Rating: 4/5)
📚 What a Good Answer Looks Like
A power of 2 in binary has exactly one set bit: 1=0001, 2=0010, 4=0100, 8=1000. The trick: n > 0 and (n & (n-1)) == 0. Why it works: n-1 flips all bits from the rightmost set bit onwards, so ANDing with n gives 0 only if there was exactly one set bit. Time O(1), Space O(1). This is the optimal and expected answer.
Count trailing zeros in n! (factorial)
Correctly identified the key insight: trailing zeros come from factors of 10 = 2 × 5; since 2s are abundant, count factors of 5 Correctly described the brute force: iterate 1 to n, count how many times each number is divisible by 5 — O(n log n) Gave the correct optimal approach: sum of floor(n/5) + floor(n/25) + floor(n/125) + ... — O(log n) Could not implement the optimal approach in code within the session
✓ Correct optimal approach stated — floor division series; implementation not completed (Rating: 4/5)
📚 What a Good Answer Looks Like
Implementation: count = 0; while n >= 5: n //= 5; count += n; return count. This divides n by successive powers of 5, accumulating contributions. Each division by 5 accounts for multiples of 5, then 25 (double-counted), 125 (triple-counted), etc. Time O(log n), Space O(1).

📚 What a Good Answer Looks Like
Dijkstra with min-heap: initialise dist = [inf] * (n+1); dist[k] = 0. Min-heap: heap = [(0, k)]. While heap: pop (d, u); if d > dist[u] skip; for each neighbour (v, w): if dist[u]+w < dist[v]: update and push. Answer: max(dist[1:]) if no inf, else -1. Time O((V+E) log V), Space O(V+E).
