📅 1-Month Action Plan
WEEK 01
HTML & CSS from Zero
  • Study all core HTML tags: headings, paragraphs, divs, spans, forms, input types (text, email, password, checkbox, radio), select/option (dropdown), tables, ordered and unordered lists, semantic tags (header, nav, main, footer)
  • Write a dropdown list from memory: <select><option>...</option></select> — repeat until automatic without any reference
  • Study CSS: box model (margin/border/padding/content), selectors (class, ID, element, pseudo), display (block/inline/flex), flexbox layout, basic grid
  • Build 3 static HTML/CSS pages from scratch: a personal profile page, a contact form with a dropdown, and a product card layout — no copy-pasting, no AI tools
  • Rule for this week: close all browser tabs except the page you are building. Write everything from memory — look at reference docs only when completely stuck, not as a starting point
WEEK 02
JavaScript from Zero
  • Study JavaScript fundamentals: variables (let/const/var), data types, functions (declaration, expression, arrow), loops (for/while/forEach), arrays (.map, .filter, .reduce), objects, DOM manipulation
  • Write 10 JS functions from memory daily — no book, no AI: greet(name), sum(a,b), reverseString(str), findMax(arr), countOccurrences(arr, item)
  • Study async JavaScript: callbacks, Promises (.then/.catch), async/await — build a demo that fetches data from a public API (e.g. JSONPlaceholder) and displays it in the DOM
  • Build a to-do list app using only HTML, CSS, and vanilla JavaScript — add, delete, and mark complete — all DOM manipulation written independently
  • Test yourself daily: close your editor, open a blank file, and write a function from memory. If you can't write it without help, you don't own it yet
WEEK 03
Flask & Backend Basics
  • Write a Flask Hello World API from scratch — independently, no AI: from flask import Flask; app = Flask(__name__); @app.route('/'); def hello(): return 'Hello World'; app.run() — repeat from memory until automatic
  • Build a simple CRUD API in Flask: GET all items, GET one item by ID, POST create item, PUT update item, DELETE remove item — use a Python list as storage first, then connect to SQLite
  • Study HTTP methods and status codes: GET, POST, PUT, PATCH, DELETE; 200, 201, 400, 401, 404, 500 — write a one-page cheat sheet from memory
  • Study MongoDB basics: collections, documents, CRUD operations (find, insertOne, updateOne, deleteOne) — create a collection and run queries in MongoDB Compass
  • Build a Flask + MongoDB API for a notes app: create note, get all notes, get one note, update note, delete note — independently, no AI assistance
WEEK 04
React Introduction & Mock Prep

  • Build a to-do list in React: add task (useState), delete task, mark complete — no copy-paste, write every line yourself
  • Connect a React frontend to your Flask backend from Week 3: fetch all notes on page load (useEffect + fetch), display them in the UI — see the full stack in action
  • Practice writing code independently: 30-minute timed sessions with no open tabs, no book, no AI — write a complete component or API endpoint from start to finish
