📚 What a Good Answer Looks Like
Boyer-Moore Voting Algorithm: maintain candidate = None and count = 0. For each element: if count = 0, set candidate = element. If element == candidate, increment count; else decrement count. Return candidate. Why it works: the majority element always outnumbers all others combined, so it survives all "cancellations". Time: O(N), Space: O(1). This is a must-know classic — memorise and understand it thoroughly.
Q2: Every number appears exactly twice except one. Find that number in O(N) time and O(1) space.

✗ Not Solved — All proposed solutions use O(N) space; XOR approach not known (Rating: 0/5)
📚 What a Good Answer Looks Like
XOR all elements: result = 0; for num in arr: result ^= num; return result a ^ a = 0 (pairs cancel out) and a ^ 0 = a (identity). All duplicate pairs cancel, leaving only the unique element. Time: O(N), Space: O(1). This appears alongside Boyer-Moore in almost every interview — know both cold.

📚 What a Good Answer Looks Like
Sort the array in descending order. Use the modular arithmetic insight: every number's remainder when divided by 3 is 0, 1, or 2. The sum of three numbers is divisible by 3 if their remainders sum to 0 mod 3 — i.e., (0,0,0), (1,1,1), (2,2,2), or (0,1,2). After sorting descending, greedily pick the three largest numbers satisfying one of these remainder combinations. Time: O(N log N) for sorting + O(1) for selection. This problem requires combining sorting with modular arithmetic — a pattern worth practising.

✗ Not Attempted — Trees not studied; no approach offered (Rating: 0/5)
📚 What a Good Answer Looks Like
Use BFS (level order traversal). Assign each node an index: root = 1, left child of node i = 2i, right child = 2i+1. At each level, the width = last node index - first node index + 1. Track the minimum index at each level to avoid integer overflow (normalise by subtracting the level's first index). Time: O(N), Space: O(N) for the queue. The key insight is using array-index-based node numbering to compute width without storing null nodes explicitly.
