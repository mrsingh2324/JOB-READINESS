📚 What a Good Answer Looks Like
Forward + Backward Kadane: build fwd[i] = max subarray sum ending at index i (standard Kadane's). Build bwd[i] = max subarray sum starting at index i (Kadane's from right). Answer without deletion = max of fwd. Answer with deletion at index i = fwd[i-1] + bwd[i+1]. Iterate over all possible deletion indices to find the maximum. Time: O(N), Space: O(N). The key insight from the hint "delete or not delete": if you delete index i, you join the best subarray ending just before i with the best subarray starting just after i — this is exactly fwd[i-1] + bwd[i+1].
