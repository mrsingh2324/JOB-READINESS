📚 What a Complete Answer Looks Like
Simple approach: str.split('').reverse().join('') — split string into array of characters, reverse in place, join back. Time: O(N), Space: O(N). Two-pointer in-place (for arrays): use left and right pointers, swap characters while left < right, advance both. Time: O(N), Space: O(1). You should be able to write both approaches fluently.
