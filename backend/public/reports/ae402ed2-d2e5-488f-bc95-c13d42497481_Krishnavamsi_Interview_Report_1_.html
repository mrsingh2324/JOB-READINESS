📚 What a Good Answer Looks Like
A power of 2 in binary has exactly one set bit (e.g., 8 = 1000). The trick: n > 0 and (n & (n-1)) == 0. Why: n-1 flips all bits from the rightmost set bit onwards; ANDing with n gives 0 only if there was exactly one set bit. Time O(1), Space O(1).

Time O(n log n), Space O(n).
